## Quick Start

<Steps>
  <Step title="Install Dependencies">
    Install Python dependencies and ensure Java 17 is available.
    ```bash
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Start Infrastructure">
    Launch Kafka and databases using Docker Compose.
    ```bash
    cd deployment/docker
    docker compose -f docker_compose.yml up -d
    ```
  </Step>
  <Step title="Run Stream Processing">
    Start the ingestion and stream processor to handle Kafka events.
    ```bash
    export PYTHONPATH=$(pwd)/src
    python -m ingestion.main
    python -m stream_processor.main
    ```
  </Step>
  <Step title="Execute Batch Pipelines">
    Run the Bronze → Silver → Gold pipelines sequentially.
    ```bash
    python -m batch_jobs.pipelines.bronze_silver.minio_to_minio
    python -m batch_jobs.pipelines.silver_silver.minio_to_clickhouse
    ```
  </Step>
</Steps>
